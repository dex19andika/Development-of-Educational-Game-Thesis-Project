Tentang Aplikasi

Aplikasi pengenalan penulisan aksara Bali ini dibuat dengan memanfaat metode CNN (Convolution Neural Network) dan Backpropagation untuk melatih data yang nantinya akan digunakan untuk menentukan tulisan pengguna apakah sudah sesuai dengan aksara yang tertera atau belum. Pelatihan pada aplikasi ini menggunakan 2070 data latih.



Aplikasi ini menggunakan bahasa pemrograman Python dengan dibantu dengan framework tensorflow dan keras, dengan library Node.js npm, Bootstrap, jQuery, dan Ajax. Untuk tampilannya sendiri dibangun dengan HTML, CSS, dan JavaScript. Dimana hasilnya ialah sebuah server lokal yang dapat menjalankan aplikasi ini
